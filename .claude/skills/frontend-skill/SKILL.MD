# Frontend Skill - Pages, Components, Layout, Styling

## Instructions

1. **Layout structure**
   - Responsive design for all screen sizes
   - Flexible grids and containers
   - Clear content hierarchy

2. **Components**
   - Buttons, cards, modals, forms
   - Reusable and consistent UI elements
   - Accessible and semantic HTML

3. **Styling**
   - CSS, Tailwind, or Styled Components
   - Use themes and consistent color palettes
   - Smooth hover, focus, and transition effects

4. **Page Building**
   - Modular structure (header, main, footer)
   - Clear spacing and alignment
   - Optimized for performance

## Best Practices
- Mobile-first approach
- Semantic HTML tags
- Reusable, maintainable components
- Consistent visual language

## Example Structure
```html
<!-- Page Layout -->
<div class="page-container">
  <header class="header">
    <h1 class="logo">My Website</h1>
    <nav class="nav">
      <a href="#">Home</a>
      <a href="#">About</a>
      <a href="#">Contact</a>
    </nav>
  </header>

  <main class="main-content">
    <!-- Card Component -->
    <div class="card">
      <h2 class="card-title">Card Title</h2>
      <p class="card-text">Card description goes here.</p>
      <button class="card-button">Learn More</button>
    </div>
  </main>

  <footer class="footer">
    <p>&copy; 2026 My Website</p>
  </footer>
</div>
